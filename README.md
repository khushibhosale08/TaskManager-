# TaskManager-
Add, Edit, Delete, Search tasks with due dates, priorities, and auto-save


## ✨ Features
- Add, Edit, Delete tasks
- Mark tasks as Complete/Incomplete
- Search tasks by keyword
- Due dates, Priority levels, and Overdue tracking
- View Statistics: Total, Completed, Pending
- Auto-save all tasks to `tasks.json`

## 🚀 How to Run
```bash
git clone https://github.com/khushibhosale08/TaskManager-.git
cd TaskManager-
python main.py
